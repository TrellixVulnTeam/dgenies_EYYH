DGENIES – for Dotplot for Genomes Interactive, E-connected and Speed – is an online tool designed to compare two genomes. It supports large genome and you can interact with the lot to improve the visualisation.

We use minimap version 2 to align the two genomes. Then, the PAF file is parsed and plotted into an interactive plot written with d3.js library.
